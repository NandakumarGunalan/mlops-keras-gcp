
# MLOps on GCP with Keras

End-to-end MLOps demo: **train → evaluate → export (.keras) → containerize → deploy to Cloud Run → monitor + drift check → retrain**.

## What this repo includes
- Lightweight Keras retraining pipeline 
- Versioned model artifacts + metrics
- FastAPI inference service
- Docker image for serving
- GitHub Actions CI (tests + build)
- GCP Cloud Run deployment workflow 
- Basic drift check job
